# LocalLink MERN Backend (localLink-backend)

This is the Node.js/Express backend for LocalLink, a Local Service Provider Directory & Booking Platform.

## ⚙️ Setup and Installation (Development)

### 1. Prerequisites

* Node.js (v18+)
* MongoDB (local or cloud instance)
* Docker (Optional, but recommended for full stack run)

### 2. Clone Repository

```bash
git clone <repository-url>
cd locallink/locallink-backend
